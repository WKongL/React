##技术栈

- 本项目采用`webpack`自动化构建工具，前端使用`react`+`react-router`+`react-Redux`，`less`预编译CSS。

##项目收获

1. react的组件化开发
2. react-Redux的基本应用

##安装与运行
- 1、npm install 安装插件  
- 2、npm run mock 启动模拟的接口数据服务器  
- 3、npm run start 可以运行项目代码
